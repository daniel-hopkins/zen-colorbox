Installation - 1.5 - 1.3.9

BACKUP

Be sure to make a backup of your database and you site files before you continue!
The BEST recommendation is to do this on a development version of your site. If you are serious about running your store it is always worth having a second version of the store so that you can experiment on that first. I know that this seems like a lot of work but it really is worth it and does not actually take that much time.
How?

BEFORE YOU INSTALL READ THIS!!!

Some files that Zen-Colorbox modifies are core files. This means that Zen-Colorbox overwrites the existing files that Zen uses. If your site has no modifications to these files then there are no problems. If your installation has existing modifications to you files then you will have to carefully merge the existing modifications with the modifications that Zen-Colorbox makes.

Best Practice

This is the way to go if you know that you have merges to do.

Read the installation instructions and see which Zen-Colorbox files are equivilant to which Zen Cart files. It is not a complex pattern! Then merge your modifications into the appropriate file in the Zen-Colorbox installation files.

For instance if you have existing modifications in
YOUR_TEMPLATE/templates/tpl_modules_main_product_image.php

merge them with the installation file
Then install as normal.

How?
To view the instructions, please choose your version of ZenCart by clicking the appropriate button:
  1. Rename the following folders to match your template folder.
    • 1_Installer_Files/includes/modules/YOUR_TEMPLATE
    • 1_Installer_Files/includes/templates/YOUR_TEMPLATE
  2. Rename the following folder to match your admin folder.
    • 1_Installer_Files/YOUR_ADMIN
  3. You can safely ignore or delete the following folder (it is only used for Zen Cart 1.5)
    • 1_Installer_Files/YOUR_ADMIN
  4. Using your FTP program; upload the includes and the (renamed) admin folder to your server (Note: All files make use of the overrides system, and should merge with your current files not replace.)
  5. Press this button to select the install sql.
  6. Press this button to select the install sql.
  7. Copy the install sql to your clipboard Edit > Copy
  8. Open a new browser window and navigate to your Store Admin > Tools > Install Sql Patches
  9. Now paste the sql that you copied in step 5 into the sql area and press send.
  10. Zen Colorbox should now be installed and ready to use. Navigate to your Store Admin > Configuration > Zen Colorbox to configure the settings

Settings

To adjust the settings for Zen Colorbox Navigate to your Store Admin > Configuration > Zen Colorbox

Uninstall - 1.5 - 1.3.9

Choose your version:
  1. Press this button to select the uninstall sql.
  2. Press this button to select the uninstall sql.
  3. Copy the install sql to your clipboard Edit > Copy
  4. Open a new browser window and navigate to your Store Admin > Tools > Install Sql Patches
  5. Now paste the sql that you copied in step 2 into the sql area and press send.
  6. Zen Colorbox should now be uninstalled from your admin area, but you will have to delete the files that were installed manually via your FTP program.
